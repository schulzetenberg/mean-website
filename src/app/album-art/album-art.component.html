Album Art Backgrounder

GitHub Project Description

Windows application built using C# and the LastFM API.

Features:

  • Set desktop background based on a artist + album
  • Set desktop background to your top LastFM albums
  • Automatically change the desktop background at a specified interval
  • Change the album artwork size
  • Change the desktop background style

    • Future Features:

  • Save album artwork to your computer
